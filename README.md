# eBay-Car-Price-Predictor

dataset from: https://www.kaggle.com/datasets/piumiu/used-cars-database-50000-data-points

Use 46504 observations, including the following features:

- price
- brand
- model
- yearOfRegistration
- odometer (km)
- powerPS (horsepower)
- vehicleType
- gearbox (manual, automatic)
- fuelType
- notRepairedDamage
- monthOfRegistration
